experimental¶

Experimental tools are not part of the test-suite, and might not be production-ready. All tools in this module must be imported via:

from toolbox.experimental import <tool>

Tools here are proof-of-concept and are not recommended to be used in production.

asyncdispatch¶

from toolbox.experimental import asyncdispatch
asyncdispatch(func: Callable) → Callable[source]¶

Decorator for adding dispatch functionality between async and sync functions.

Similar to functools.singledispatch(), but for sync and async function. This decorator allows for a single function name to be used for two different implementations, one synchronous and another asynchronous. Note that in the implementation of asyncdispatch() the inspect module is utilized, and therefore might not work with different implementations of Python.

Parameters

func – Synchronous function to create a dispatch with.

Warning

Not recommended to be used in production in its current implementation.

Example

from toolbox.experimental.asyncdispatch import asyncdispatch
import asyncio

@asyncdispatch
def func():
    return "sync"

@func.register
async def _():
    return "async"

async def main():
    print(func())          # >>> sync
    print(await func())    # >>> async

asyncio.run(main())
